# 🐰 IT研修で使える英語フレーズ集を作ったのでシェアします

オンライン研修で英語を使う場面が増えてきたので、「サッと確認・コピー・発音練習できる」ツールを自分用に作りました。

**IT研修に関わる講師・トレーナーの方**、よければ使ってみてください。

---

## どんなツールか

**うさうさ研修工房 ｜ FOR IT TRAINERS ｜ v2.0**

Zoom・Slack・コーチング・試験運営など、研修現場で使う英語フレーズを22タブにまとめたブラウザツールです。各フレーズにカタカナ読み・日本語訳・📋コピー・🔊ネイティブ発音がついています。

| | |
|---|---|
| 動作 | HTMLファイルをブラウザで開くだけ |
| 推奨ブラウザ | **Chrome** |
| インストール | 不要 |
| データ | 端末内のみ保存・外部送信なし |
| オフライン | 可（発音フィードバックのAI機能を除く） |

---

## 22タブでできること

### 🗣 ツール・コミュニケーション

| タブ | 内容 |
|---|---|
| ① ツール別 | Zoom / Google Meet / oVice / Slack でサッと言える・貼れる英語。📋コピー・🔊発音 |
| ② 報連相・提案 | チャットやメッセージにそのまま使える、やわらかい言い方 |
| ③ 確認（理解・設計） | 「合ってますか？」を角を立てずに。設計レビューにも |
| ④ MTG・アナウンス | 講師ミーティングの進行と、受講生への連絡テンプレ |

### 📚 レッスンフレーズ

| タブ | 内容 |
|---|---|
| ⑤ Pythonレッスン | 基礎〜応用（関数定義・例外処理・pandas）。受講生の質問／講師の説明を目線トグルで切替。📋コピー・🔊発音 |
| ⑥ LPICレッスン | 基礎＋科目別（権限・プロセス・ネットワーク・シェルスクリプト）。目線トグルで受講生／講師を切替。📋コピー・🔊発音 |

### 🚨 現場対応

| タブ | 内容 |
|---|---|
| ⑦ トラブル時 | 接続が切れた・画面が固まった・環境に入れない…そんな時の落ち着いた言い方 |
| ⑧ つまずき先回り | 受講生がつまずく前に安心させる・予告する講師の一言。否定せず、前向きに |
| ⑨ 進捗・日報 | 受講生は自分の進捗を、講師はクラスの状況・日報を。目線トグルで切替 |

### 🤝 指導・コーチング

| タブ | 内容 |
|---|---|
| ⑩ 研修ビジネス | カリキュラム設計・経営層ヒアリング・フォローアップ・継続支援・MTGの英語 |
| ⑪ コーチング | 否定せず問いかけで導く。Why／What／Howで理解確認、ヒント、「惜しい！」の言い方 |
| ⑫ 学習ヒント | つまずいた受講生に渡す"つかみ方"のヒント。文系・理系の得意に合わせて |

### 🎤 発音練習

| タブ | 内容 |
|---|---|
| ⑬ 発音フィードバック | お手本を聞く→話す→AIがやさしく添削。マイクが使えない時は手入力でも可（AI機能） |
| ⑭ 暗記テスト | 選んだ範囲から4択で出題。意味↔英語を切替可。まちがいは"伸びしろ"扱い 🐰 |

### 👥 受講生・講師それぞれのビュー

| タブ | 内容 |
|---|---|
| ⑮ 受講生向け | 全カテゴリから受講生フレーズだけを集約。カテゴリ絞り込み・分野タグつき |
| ⑯ 講師が使う言葉 | 全カテゴリから講師フレーズだけを集約。コーチング・MTG・ビジネス・試験運営まで横断一覧 |
| ⑰ 回答FB | テスト前プレビュー／テスト後の結果まとめを切替表示 |

### 📝 試験・評価運営（心理的安全に配慮）

| タブ | 内容 |
|---|---|
| ⑱ 口頭試問 | 合否で脅かさない。「わからない」も歓迎。安心して話せる口頭確認の言い回し |
| ⑲ 筆記試験 | 開始・指示・見直し・終了の進行。部分点や減点なしを伝えて安心させる |
| ⑳ 理解度クイズ | 成績でなく「どこをゆっくり進めるか」を見るため。間違いは講師の説明し直しサイン |
| ㉑ 解答・採点 | 正解の確認、部分点、よくある間違いの共有。考え方をほめてから直す |
| ㉒ 時間・進行 | 残り時間の告知、ペース調整、予告ワーニング。レッスンを気持ちよく回す |

## 入手・カスタマイズ

**HTMLファイル版**（推奨）— 本記事に添付のファイルをChromeで開けばすぐ使えます。

```
【PRO】うさうさ研修工房 ｜ FOR IT TRAINERS ｜ v2.0 ｜ 2026.0615.html
```

**アーティファクト版**（カスタマイズしたい方向け）— Claude.ai のアーティファクト機能でHTMLを貼り付けると、「このフレーズを追加したい」「タブを改造したい」など話しかけながら自由に編集できます。※Chrome推奨
